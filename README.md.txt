#Ping Pong Game

##Introduction 
- This is a very simple Ping Pong game project, developed after I had already
completed module "Introduction to Programming" in the first semester at university.

##Features
- 2-player game
- Basic GUI using JAVA's libraries including AWT and Swing
- Simple operation with ball movement and collision detection
- Writed by IntelliJ IDEA and Object-Oriented structure : separate class for Ball,
Player, ControlWindow

##How to run
1. Clone from my github link
2. Compile the source files

Note: It will be redesigned as a web-based game for people can easily get access
to it.
